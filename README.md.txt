Sistema Analítico de Concessionária

Projeto de modelagem relacional e análise de vendas utilizando MySQL.

Objetivo

Simular o banco de dados de uma concessionária para realizar análises comerciais como:

- Faturamento total
- Faturamento por marca
- Cliente com maior ticket
- Marca mais vendida
- Ticket médio

Tecnologias

- MySQL
- Modelagem Relacional
- JOIN
- GROUP BY
- Views
- Stored Procedures

Principais Insights

- Identificação da marca com maior faturamento
- Cliente com maior volume de compras
- Análise de ticket médio

Estrutura

- database.sql
- queries_analiticas.sql
- views.sql
- procedures.sql
